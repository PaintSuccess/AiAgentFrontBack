# Cancellation/refund note templates

## Standard

```text
Daniel, reminder: {customer} {source} requesting cancellation and refund for order {order_number}. Please cancel/refund this order.
```

## With product

```text
Daniel, reminder: {customer} {source} requesting cancellation and refund for order {order_number} ({product}). Please cancel/refund this order.
```

## With unknown customer

```text
Daniel, reminder: customer requested cancellation/refund for order {order_number}. Please review and cancel/refund if appropriate.
```

## Staff-facing rules

- Keep notes short and actionable.
- Include source: emailed, called, spoke by phone.
- Do not promise refund completion unless actually completed.
